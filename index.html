💡 Paramètres disponibles:
  • pollinationsKey — Votre clé API Pollinations (sk_...)
  • googleWebhook — URL du webhook Google Apps Script pour les RDV
  • businessName — Nom affiché dans le header du chat
  • welcomeMessage — Message de bienvenue affiché au premier ouverture
  • color — Couleur principale (hex, ex: "#6366f1")
  • position — Position du widget: "right" ou "left"
  • systemPrompt — Instructions pour personnaliser le comportement de l'IA

🗓️ Google Calendar Setup

Connecter Google Calendar

En 5 minutes, vos RDV s'enregistrent automatiquement dans votre agenda Google.

1

Ouvrir Google Apps Script

Allez sur script.google.com et créez un nouveau projet.

2

Coller ce code dans le fichier Code.gs

function doPost(e) {
  var data = JSON.parse(e.postData.contents);

  // ── GOOGLE CALENDAR ──
  var cal = CalendarApp.getDefaultCalendar();
  var dt  = new Date(data.datetime);
  var end = new Date(dt.getTime() + 60 * 60000); // 1h
  cal.createEvent(
    "RDV: " + data.name + " — " + (data.service || "Service"),
    dt, end,
    { description: "Tel: " + data.phone + "\nVia AgentChat" }
  );

  // ── GOOGLE SHEETS ──
  var sheet = SpreadsheetApp.openById("VOTRE_SHEET_ID").getActiveSheet();
  sheet.appendRow([
    new Date(), data.name, data.phone,
    data.datetime, data.service, data.businessName
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({ success: true }))
    .setMimeType(ContentService.MimeType.JSON);
}
3

Déployer comme Web App

Cliquez sur Déployer → Nouveau déploiement, choisissez "Application Web", définissez l'accès sur "Tout le monde" et copiez l'URL obtenue.

4

Coller l'URL dans votre config

Mettez l'URL dans googleWebhook dans votre code d'intégration. C'est tout! ✅
